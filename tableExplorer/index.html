Information

Generally in character matrices, each row of the matrix corresponds to a separate species and the columnsof the table correspond to individual characteristics (or "characters") observed in the species. At the bottom of the window, each characteristic is listed. The user can disable a column by clicking the 'x' next to the column name, restore a column by reselecting the tag, and rearrange the columns by directly dragging the tags into the order desired. Clicking the heading on a column sorts the matrix accordingly. Entering text in the "Filter" box will mask the matrix so that only rows containing matching text will be displays (like the unix "grep" command).
